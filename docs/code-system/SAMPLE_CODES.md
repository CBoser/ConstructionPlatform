## Pattern Templates

### Lumber Pattern
```
FRM-LUM-[SIZE]-[SPECIES]-[LENGTH]
Example: FRM-LUM-2X6-SPF-12
```

### Engineered Wood Pattern
```
FRM-ENG-[TYPE]-[DEPTH]
Example: FRM-ENG-LVL-11.875
```

### Sheathing Pattern
```
FRM-SHT-[TYPE]-[THICKNESS]-[SIZE]
Example: FRM-SHT-OSB-7/16-4X8
```

### Roofing Pattern
```
ROOF-[TYPE]-[STYLE]-[WARRANTY/COLOR]
Example: ROOF-SHG-ARCH-30Y-BLK
```

### Siding Pattern
```
SIDE-[MATERIAL]-[PROFILE]-[SIZE]
Example: SIDE-FBC-LAP-8.25
```

### Insulation Pattern
```
INSL-[TYPE]-[R-VALUE]-[WIDTH]
Example: INSL-BAT-R19-15
```

---

## Notes

- Codes are case-insensitive (stored uppercase)
- Maximum 25 characters
- Use hyphens as separators
- Details are optional but recommended for clarity
- DART column maps to BAT pricing system categories
